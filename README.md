This is a simple PG listing web application build using HTML, CSS, Javascript.
It design PG  details like name, price, room type, gender, and availability.
Users can filter PG's based on price, room type, gender,and availability.
Users can mark favoriate PG's using heart icon.
Favourite data is stored using local storage.
The Price slider is used to filter the PG's dynamically.
PG's listing is shown in the foem of cards.
The UI updates automatically based on the selected filters.
To run open the index.html file in any web brower.
The application will start running directly in the brower.
